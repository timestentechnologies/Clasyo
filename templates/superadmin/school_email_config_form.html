Configuration Help
SMTP Configuration

Configure your SMTP server for email delivery.

  • Host: SMTP server address (e.g., smtp.gmail.com)
  • Port: Usually 587 (TLS) or 465 (SSL)
  • Security: Choose TLS, SSL, or None
  • Username/Password: Your SMTP credentials
SendGrid Configuration

SendGrid is a cloud-based email service.

  • Get API Key from SendGrid dashboard
  • Ensure sender is verified in SendGrid
  • API Key should have "Mail Send" permissions
Mailgun Configuration

Mailgun provides developer-friendly email APIs.

  • Get API Key from Mailgun control panel
  • Domain must be verified in Mailgun
  • Use sandbox domain for testing
Amazon SES Configuration

AWS Simple Email Service for scalable delivery.

  • Create IAM user with SES permissions
  • Verify your sending domain in SES
  • Move out of sandbox mode for production
Postmark Configuration

Postmark offers fast and reliable email delivery.

  • Get Server Token from Postmark dashboard
  • Verify sender signatures
  • Monitor delivery statistics

Select an email provider to see configuration help.

Best Practices
  • Always use secure connections (TLS/SSL)
  • Test configuration before enabling
  • Set reasonable usage limits
  • Monitor email delivery rates
  • Keep API credentials secure
  • Use custom from email for branding
