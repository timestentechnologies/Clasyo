Configuration Guide
M-Pesa Setup
  1. Register on Safaricom Developer Portal
  2. Create a new app and get Consumer Key & Secret
  3. Request for Lipa na M-Pesa Online Shortcode
  4. Generate and download the security certificate
  5. Test with sandbox credentials before going live
PayPal Setup
  1. Create a PayPal Business Account
  2. Go to PayPal Developer Dashboard
  3. Create a new REST API app
  4. Copy Client ID and Client Secret
  5. Set up webhooks for payment notifications
Stripe Setup
  1. Create a Stripe Account
  2. Get API keys from Dashboard → Developers → API keys
  3. Test with test keys before using live keys
  4. Configure webhooks for payment events
  5. Set up payment methods in Stripe Dashboard
Bank Transfer Setup
  1. Contact your bank for business account details
  2. Get account name, number, and branch information
  3. Ask for SWIFT code for international transfers
  4. Display bank details to customers for manual transfers
  5. Verify payments manually through bank statements
Getting Started

Select a payment gateway from the dropdown to see configuration instructions and required fields.

Always test payment configurations in sandbox mode before going live.
