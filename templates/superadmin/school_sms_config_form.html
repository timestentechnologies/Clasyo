Configuration Help
Twilio Configuration

Twilio provides global SMS delivery with detailed analytics.

  • Get Account SID and Auth Token from Twilio Console
  • Purchase a Twilio phone number for sending SMS
  • Ensure your account has SMS capabilities
  • Format phone numbers with country code (+1234567890)
Africa's Talking Configuration

Africa's Talking specializes in African market SMS delivery.

  • Sign up at africastalking.com
  • Get API Key from your dashboard
  • Username is typically your app username
  • Supports local payment methods in Africa
Infobip Configuration

Infobip offers omnichannel communication with advanced routing.

  • Create account at infobip.com
  • Get API Key from your portal
  • Default base URL: https://api.infobip.com
  • Supports advanced message personalization
Clickatell Configuration

Clickatell provides enterprise-grade SMS solutions.

  • Register at platform.clickatell.com
  • Generate API Key from your account
  • Default base URL: https://platform.clickatell.com
  • Offers compliance and reporting features
Vonage (Nexmo) Configuration

Vonage (formerly Nexmo) provides cloud communication APIs.

  • Create account at vonage.com
  • Get API Key and API Secret from dashboard
  • Supports virtual numbers for sending
  • Offers voice and messaging APIs

Select an SMS provider to see configuration help.

Best Practices
  • Keep API credentials secure and private
  • Test with small volumes first
  • Set reasonable usage limits to control costs
  • Use recognizable sender IDs (max 11 chars)
  • Monitor delivery rates and failed messages
  • Have backup SMS providers configured
Sender ID Guidelines
  • Alphanumeric: Max 11 characters (A-Z, 0-9)
  • Numeric: Max 15 digits
  • No spaces: Use underscores or no spaces
  • Brand recognition: Use your school name
  • Country-specific: Some countries have restrictions
